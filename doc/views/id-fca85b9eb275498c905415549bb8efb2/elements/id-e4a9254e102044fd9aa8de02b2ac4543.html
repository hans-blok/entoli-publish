Policy (scope) ()
A policy (scope) is a set of explicit rules, constraints, and permissions that governs what an Entoli agent is allowed to do within a defined workspace scope. It determines which artifacts may be accessed, which operations are permitted, and under which conditions the agent may act. A policy (scope) is applied by the agent at runtime to enforce governance boundaries.
