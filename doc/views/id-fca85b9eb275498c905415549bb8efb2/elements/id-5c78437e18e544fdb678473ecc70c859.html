Origin Position ()
Origin Position is the classification axis that specifies the initiating context of a business artifact — whether it is created from scratch to start a new thread of work, or built on top of an existing artifact. It records whether an artifact is an initiating artifact (starting a new line of work) or a building-on artifact (extending, refining, or transforming prior work). POSITIONS ORIGIN POSITION - Initiating — the artifact originates a new thread of work without a prior artifact of its type as input - Building-on — the artifact is explicitly derived from or extends an existing artifact
