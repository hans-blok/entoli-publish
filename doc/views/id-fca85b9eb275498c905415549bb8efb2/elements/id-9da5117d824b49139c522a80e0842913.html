Entoli-artifact ()
An artifact is an explicitly created, durable work product produced or consumed by an Entoli agent within a project workspace. An artifact is either a normative artifact (prescribing rules or norms) or a business artifact (recording content, decisions, or deliverables). Artifacts are the primary units of value exchange within a value stream phase. NOTE: the "normative vs. business" split used here is not the canonical sub-classification — canon classifies Entoli-artifact via the artifact-function-axis (Governance / Directive / Implementing / Structuring / Registering artifact); see alignment report.
