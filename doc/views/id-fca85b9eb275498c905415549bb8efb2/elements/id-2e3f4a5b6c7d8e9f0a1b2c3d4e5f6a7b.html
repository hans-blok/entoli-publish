Mode of operation ()
Mode of operation is a classification axis that describes whether an entoli-agent intervenes in content or in the conditions under which content work can take place. POSITIONS - Content — the agent operates primarily on the substantive content of its input - Representation-transforming — the agent transforms the form or format of existing content - Conditional — the agent applies rules or conditions to determine its output
