Entoli-concept ()
An Entoli concept is the ecosystem-level synonym for Element. Every explicitly defined, coherent and reusable unit of meaning within the Entoli ecosystem is an element in the semantic graph of the canon. The term Entoli-concept is used to refer to any such element by name within ecosystem communication.
