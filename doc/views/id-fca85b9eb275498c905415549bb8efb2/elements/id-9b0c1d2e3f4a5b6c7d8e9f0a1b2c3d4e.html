Source-regime ()
Source-regime is the classification axis that specifies, per agent-contract, which source space the agent may use during an execution, and thus how strictly derivation is bounded. The source-regime records the concrete rules for that position: which sources are valid, which are excluded, and whether interpretation or synthesis is permitted.
