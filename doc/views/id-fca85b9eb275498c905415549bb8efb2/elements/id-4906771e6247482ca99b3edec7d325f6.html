Source-regime ()
Source-regime is the classification axis that specifies, per agent-contract, which source space the agent may use during an execution, and thus how strictly derivation is bounded. The source-regime records the concrete rules for that position: which sources are valid, which are excluded, and whether interpretation or synthesis is permitted. SOURCE REGIME - Input-bound — the agent works from explicitly provided input - Canon-bound — the agent draws from the canonical foundations - Workspace-bound — the agent draws from workspace artefacts - Externally-bound — the agent draws from an external source (web, API) - Exploratory — the agent explores without a fixed source regime
